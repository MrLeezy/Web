内置插件 Data Ops

商机数据整合

上传不同来源的商机 Excel，系统会自动识别 website / Chat / 400 / order，并在同一工作区里累加整合，最终生成一个标准终极表。

工作区文件 0 当前批次已上传文件
可构建文件 0 已识别来源且可参与生成
未识别文件 0 需要你先确认是否上传了正确文件
最近输出 暂无 最近一次生成的终极表

步骤 1

上传并识别来源文件

支持多选上传。系统会自动按文件名、sheet 和表头识别来源类型，并把所有文件累加保留在当前工作区中。

先上传文件，再点击生成终极表。

系统状态

规则资产检查

终极表字段顺序由映射表控制，channel 口径由渠道映射表控制。

当前工作区

这里会展示每个上传文件的识别结果。`unknown` 文件不会参与构建，你可以删除后重新上传正确文件。
